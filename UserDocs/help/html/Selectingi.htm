Selecting

To 'permanently' select one or more objects:

  1. point the mouse arrow at the empty screen area
  2. click-and-drag over object(s) you want to select
  3. release the mouse button when you finished

Selected objects will be blue colored. Also, four stretching handles (red squares) will be displayed at selection corners.

Operatios on selected objects

  • moving - mouse click-and-drag at any of selected objects to move the selection
  • stretching - mouse click-and-drag at any of four red stretching handles
  • rotating - mouse righ-click-and-drag at any of four red stretching hanldes
  • cut, copy, paste - use Ctrl+X, Ctrl+C, Ctrl+V (or use the File->Edit menu)
  • aligning, mirroring, grouping... - right-click at any of selected objects to open the context menu

Adding and excluding items to/from the selection

You can add to an existing selection by holding the shift key while selecting additional objects. You exclude objects by holding the shift key and mouse-clicking at already selected objects.

An alternative way to select/deselect an object is by pointing mouse on it and then using the Alt+SpaceBar keystroke.

To deselect everything, hit the Esc key, or just mouse click at empty document area.
