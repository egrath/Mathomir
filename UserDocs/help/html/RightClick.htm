Right mouse click

  • right mouse click at a touched object will open the Context Menu.

  • right click at an Insertion point is used for ‘quick multitouch’ feature. All elements at the same operator precedence level are touched; the Context menu is then opened.

  • right mouse click at empty document area can: clear selection, throw away carried object, refresh the screen, close popup menus, end the keyboard entry mode, end hand-drawing mode… It performs similar to pressing the Esc key.

  • long right-click will show the handy hand-drawing-popup toolbox and start the hand-drawing mode

  • right click-and-drag will scroll the entire document area (similar to using scroll bars)
