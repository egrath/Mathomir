Generating equation image

You can copy equation imagees (bitmaps) into some other program over Windows Clipboard. Either touch or select the object whose image you want to generate and then press the F9 key or Ctrl+C. After that you can paste the image into another program.

You can also directly write the image into file (PNG, JPEG, BMP). In this case, use the F8 key.

You can control the resulting image using ‘Options->Output image’ menu.

LaTeX

You can generate LaTeX code in a similar way. Hit the F6 key (or choose ‘Edit->Copy LaTeX code’) and the code will be copied into the windows clipboard. You can then paste it into any text document.
