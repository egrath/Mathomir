Entering a plain text

To enter a plain text you need to activate the text-typing mode.

  • ALT+Spacebar toggles between math-typing and text-typing mode.
    (Alternatively, you can configure the CapsLock key to be used for the same purpose)

The color of the blinking cursor indicates what typing mode is currently active:

  • Blue cursor - the math-typing mode is active
  • Green cursor - the text-typing mode is active
    •

Alternative ways to start plain text entry

  • while a box is still entirely empty you can use the spacebar key (in addition to Alt+spacebar) to toggle the typing mode
  • you can mouse-click at the sigma sign in the text-control box to switch into text-typing mode (the text-control box will change its color to green)
  • instead of using single mouse click to start a box, you can double-click to start a box that already has the text typing mode activated
  • you can define a text guideline - by clicking on it, the new box is started and will have the text typing mode activated
  • you can just start typing plain text even if the math-typing mode is still active - after several keystrokes Math-o-mir will revert into text-typing mode (this only works if your text entry started from an empty box, if your typing speed is sufficient and if you don't use some math operators)
