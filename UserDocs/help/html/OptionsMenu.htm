The Options menu

Selections – choose additional method to highlight touched objects.

Moving dot – choose the size and behavior of the Moving Dot

Mouse - choose mouse and mouse-wheel preferences

Keyboard – you can choose keyboard behavior:
‘Fix font for numbers’ will force all typed numbers (0…9) into the same font (Arial).
‘Alt for menu shortcuts’ enables ALT+F, ALT+E, ALT+V to access main menu.
'F1 sets zoom level to 100%' - redefines the F1 key (by default it loads the handy-help)
'Use CapsLock to togle typing mode' - redefines the CapsLock key (Shift+CapsLock will toggle uper/lowercase)
'Allow comma as decimal separator' - enables you to wirite pi as 3,14 (by default 3.14 is only enabled)
'Use complex indexes' - enables placing math operators into indexes
'Prefer Multi-Letter variables' - multi-letter variables are easily typed
'Prefer Single-Letter variables' - single-letter variables are easily typed

Output image – you can choose the size and quality of the output equation image.

Font size – choose default font size for new equations you start typing. You can also redefine font faces to be used with Math-o-mir.

Parenthese height – chose the ‘default’ height of parentheses you enter into equations

Grid and guidelines – defines the grid resolution. You can also enable snap-to-grid and snap-to-guidelines

Symbolic computation – enables/disables the symbolic calculator, defines various symbolic calculator options.

Save settings – you can save menu options and toolbox settings to disk (whether into the default file – mathomir.set – or into your own file). You can also load previously saved options.
