Math-o-mir in a classroom

This software can be used by teachers and/or students as a smooth way to replace paper-and-pencil.

In addition, it is possible to make electronic math exams with Math-o-mir.
To save a file as an exam, go to File->SaveAs, then under ‘Save as type’ choose option ‘MOM exam file’. You will be asked to provide password and time limit for the exam. The password is not used to open the exam file, but is used to open result files stored by students. Only teacher that knows the password will be able to see results.
