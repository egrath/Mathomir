Entering commands, units and functions

To type a command, you must start with the ‘\’ (backslash) character. The command is executed by pressing the space bar key, the enter key or by entering any math operator.

Some of commands are:

  • \lim – inserts limes
  • \int, \iint, \iiint – inserts integrals (single, double, triple)
  • \oint, \oiint, \oiiint – inserts circular integrals (single, double, triple)
  • \sum – inserts summation
  • \frac – inserts fraction
  • \binom – inserts binom
  • \sqrt, \root – inserts square root and general root
  • \case – inserts condition list
  • \alpha...\omega, \Alpha... \Omega – inserts a Greek symbol
  • \d, \dx, \dy, \dz, \dt, \par, \partial, \dd, \ddx, \ddy, \ddz, \ddt – inserts differentials
  • \funcname – inserts function of the given name (example: \funcsum)
  • \unitname – inserts unit of the given name (example: \unitpeso)
  • \e, \i, \j – Euler’s number, imaginary units
  • \big, \bbig, \small, \red, \green, \blue, \gray, \black, \vert – change size and color
  • \h1, \h2, \h3 - converts to headline
  • \link, \label - inserts link or label objects
  • \\ - (double backslash) starts editing new equation, just below this one

Entering functions

You can enter any function by typing command: \function_name, where the function_name can be any string that is not already reserved as a command or a measurement unit. Example: '\sin'

Still, the fastest way to enter a function is by 'conversion' method: type the function name and then hit the dot (period) key to convert it into function. Example: 'sin.'

Enterning units of measurement

To type a unit, start with the dot (period) character. For example '.kg'... Measurement units are displayed in a pale color.

Special cases: .deg (° - degree), .degC (°C), .ohm (resistance). Just after a number, it is possible to write degree of arc by typing '''. It is also possible to enter degree of Celsius by typing 'C.
