Edit equations using keyboard

To start the Typing mode either:

  • mouse click at empty document area - starts the new equation
  • mouse click at an insertion point – to edit the existing equation

When in the Typing mode:

  • the mouse arrow is changed into a heart-like black shape
  • the keyboard control box is displayed below the toolbox
  • you cannot carry any expression/equation! – as soon as you pick it up, it is automatically transferred and placed at the keyboard cursor position.

To exit the Typing mode either:

  • right click anywhere on empty document area
  • press the Esc key

Note: you don't even have to start the Typing mode to type in a letter or two. You can use the quick-typing feature instead - just point your mouse pointer where you want and start typing (hit the space bar first if you want to enter plain text instead of math).
