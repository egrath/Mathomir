General tips for typing

  • Alt+spacebar - toggles between math or text typing mode.
  • letter double-strokes – produce Greek symbols. ('aa' produces 'alpha').
  • Alt+key - adds characters into exponent.
  • SpaceBar+key - adds characters into index.
  • ^ or " - starts the exponent entry
  • _ (underline) - starts the index entry
  • Shift+arrow_key – selects objects relative to cursor position (see Working with selections)
  • Ctrl+key – user editable Toolbox accelerators. For example, Ctrl+R may generate root.
  • Tab – adds a spacer character (its length can be mouse-adjusted)
  • Shift+Tab – adds a column (matrices)
  • Shift+Enter – adds a row (matrices)
  • Alt+Enter - forces line wrap
  • <<,>>, <=, >=, +-, -+, ->, +>, :=, =:, /=   – these sequences are turned into simple operators
  • [ ), ( ], < >, [>, <]- these sequences are turned into appropriate brackets
  • // (two slashes) – same as Alt+spacebar
  • # – inserts a simple fraction (inserts a fraction line under the last typed object)
  • Alt+arrow_key - produce arrows. Repeat twice for double arrow.
  • @ - turned into the infinite sign
  • * – turned into small dot
  • \* - inserts the * operator
  • \# - inserts the # sign
  • \{ - inserts left cruly bracket
  • \} - inserts right curly bracket
  • Alt+, - adds an arrow over the variable (hit twice to add a dash)
  • Alt+. - adds an dot over the variable (hit twice to add a double-dot)
  • ^^ - adds a hat over the variable
  • ’ (single quotation mark) – used as a way to start writing multi-letter variables.
  • SpaceBar+arrow_key - adjusts equation position.
  • SpaceBar+dot - paste from auto-clipboard
