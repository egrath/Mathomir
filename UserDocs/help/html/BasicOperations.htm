Basic mouse operations

There are four basic mouse operations: Touch, Pick-up, Carry, Place-down

  1. Touch an object (the square root in this example)
  2. Pick up its copy (left mouse click)
  3. Carry it wherever you want
  4. Touch where you want to place it down (the insertion point between ‘14’ and ‘z’ in this example)
  5. Place-down the carried object by left mouse click

Touching – aim the mouse pointer at an object and it will highlight blue. By pointing at right spots you can touch whole expressions and even whole equations...

  • To touch more than one object (multi-touch), hold down the Shift key (or mouse button) and move the pointer over the equation.
  • You can also touch insertion points. Touched insertion point is represented by blue or green line (as in the step 4. of the example above).

Picking-up and carrying – once you touched something, mouse-click to pick up a copy of it. The copy can then be carried away with your mouse.

  • The mouse arrow becomes black when you carry anything.
  • You can always throw away whatever you carry by pressing the Esc key, or by right-click at empty document area.
Note: Instead of copy, you can cut touched object by holding the mouse button down for a longer time (a long-click) while you performing the pick-up action. The effect is the same as if you used Ctrl+X (cut).

Placing-down – you can place down the carried object by mouse click. There are several distinguished ways to place-down:

  • Over other object(s)– touch object(s) while you carry something and then mouse-click. The touched object(s) will be replaced
  • At an insertion point – touch an insertion point while you carry something and then mouse-click. The carried objects are inserted.
  • At empty document area – point the mouse button at empty area while you carry something and mouse-click. A new object is created.

The Insertion Point

The Insertion point is a place (usually between two elements in an equation) where you can insert new elements. When touched, the insertion point is visible as a blue or green line. Two things can happen if you left-mouse-click at an insertion point:

  • if you carried something, this will be inserted here
  • if you didn’t carry anything, the keyboard entry mode will start
