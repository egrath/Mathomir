Drawing boxes

You can place a drawing box into your document for easier technical drawing. The Drawing box displays a drawing toolbox and coordinate grid when cursor is inside it.

By mouse clicking at ‘>’ symbol in the upper left corner, you can enter few basic commands:

line length[,angle]
line x1,y1,x2,y2
vline length[,angle]
rect width[,height]
rect x1,y1,x2,y2
ellipse width[,height]
ellipse x1,y1,x2,y2
rotate angle
move x[,y]

the following commands act on the latest object drawn:

red, green, black, blue, gray
dashdash, dashdot
hair, thin, thick, medium

For example vline 10,30 will draw line of length=10 and angled for 30 degrees. You can enter several commands at once by separating them with the ‘;’ character. For example: line 10,10,15.3,10;thin

You can move the origin point and define unit lengths by mouse click-and-drag action on the origin point and unit-length marks.

When plotting inside the drawing box, you can hold the ALT key to activate snap-to-point function. You can snap to line end-points and line intersections.
