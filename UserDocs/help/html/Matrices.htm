Matrix editing

Matrices are supported in Math-o-mir. Some empty matrices and vectors can be found in the main toolbox (under the ‘parentheses’ subtoolbox).

You can also construct a matrix or vector with keyboard... use the Shift+Tab key to insert column separator, and Shift+Enter keystroke to insert row separator. An alternative way to insert rows and columns would be by mouse-clicking at appropriate buttons in the text-control box.

Once a matrix or vector is generated, you can use mouse to edit it further:

  • click at the row insertion line or column insertion line to add the new row or column
  • touch all elements in row (or column) holding the shift key – you can then delete that row (column) by pressing the ‘Del’ key or Ctrl+X or by choosing ‘Delete’ from the context menu.
  • Pick up some sub-selection of the matrix – you can place it down over already existing cells or insert it between rows/columns.

A matrix can have maximum 50 rows and maximum 50 columns.
